## How to Join "wiz_simu" Project
当プロジェクトに参加したい方は、[@wiz_tools](https://twitter.com/wiz_tools)にリプライまたはDMを送るか、  
<master@wiztools.net>にメールをお送りください。手順をお伝えします。  
もちろん、cloneしてpull requestを投げつける方法でもOKです。
